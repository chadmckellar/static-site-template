# Static Site Template

## Description

This repository serves as a template for static websites. Clone or fork to get started quickly on your next web project.

## How to Use

1. Clone this repo
2. Install dependencies: `npm install`
3. Start development server: `npm start`
4. Build for production: `npm run build`
